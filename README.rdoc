== DBC Freehub

The Durham Bike Co-op (DBC) Freehub is a Ruby application used by DBC to track volunteer hours, achievement
progress, and event attendance. DBC Freehub is forked from a similar Ruby application created by the San
Francisco Bike Kitchen located at https://github.com/asalant/freehub.

DBC Freehub is built using the Rails framework and is deployed on Digital Ocean at TBS.

The DBC Freehub home is https://github.com/durhambikecoop/freehub. Project details, user and developer
guides, and the latest news are located at https://wiki.github.com/durhambikecoop/freehub/.
